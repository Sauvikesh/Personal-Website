Here are some projects I've worked on.

BattleShip

Created the game “Battleship” in C++ l, but implemented a set of AIs allowing the user to play human vs human, human vs AI, and AI vs AI. The AI had 3 different versions: random AI, cheating AI, and hunt/destroy AI that can be selected when starting the game. Printed the players firing and placement board each turn in the terminal. Utilized object-oriented building techniques and polymorphism to organize the code across 29 files.

Snake

Created the game "Snake" in C++. The game prints to the terminal and takes in user input one at a time instead of making the snake move forward automatically. Employed object-oriented program-building techniques to organize code.
